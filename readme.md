# How to Run

1. Install necessary libraries:
    - PyTorch
    - Transformers
    - Pandas
    - Jupyter Notebook

2. Run the `.ipynb` by the `STEP` index. (Step descriptions are in the Notebooks. Please read with the report (In fact, you are recommanded to read the report first.).)
